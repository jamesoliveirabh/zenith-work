# Zenith Work

Aplicação web para gestão de tarefas, documentos e automações de equipes.

## Como executar

1. Instale dependências:
   ```bash
   npm install
   ```
2. Rode o ambiente de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Rode os testes:
   ```bash
   npm test
   ```
